How To Installasion
1. Clone this repository
2. Run "composer install"
3. Copy **.env.example** to **.env**
4. Run "php artisan key:generate"
5. Configuration database in **.env**
6. Create database
7. Run "php artisan migrate --seed"
8. To Start run "php artisan serve"
